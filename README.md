# 顔文字屋 Chrome Extension

簡単に顔文字を検索してコピーできる便利な Chrome 拡張機能です。

## 🚀 インストール方法

### 1. アイコンの生成

`icon-generator.html` をブラウザで開いて、3つのアイコン（16x16、48x48、128x128）をダウンロードしてください。

または、開発テストの場合は manifest.json からアイコンの設定を一時的に削除することもできます。

### 2. Chrome に拡張機能を読み込む

1. Chrome ブラウザを開く
2. アドレスバーに `chrome://extensions/` を入力
3. 右上の「デベロッパーモード」をオンにする
4. 「パッケージ化されていない拡張機能を読み込む」をクリック
5. このフォルダ（kaomoji）を選択

### 3. 使用方法

1. Chrome ツールバーの拡張機能アイコンをクリック
2. 検索バーでキーワード検索、またはカテゴリーを選択
3. 顔文字をクリックしてコピー
4. 「もっと見る」をクリックすると kaomojiya.org に移動します

## 📁 ファイル構成

```
kaomoji/
├── manifest.json       # 拡張機能の設定ファイル
├── popup.html          # ポップアップUI
├── popup.css           # スタイルシート
├── popup.js            # メインロジック
├── data.js             # 顔文字データベース
├── icon-generator.html # アイコン生成ツール
├── icon16.png          # 16x16 アイコン（生成が必要）
├── icon48.png          # 48x48 アイコン（生成が必要）
└── icon128.png         # 128x128 アイコン（生成が必要）
```

## ✨ 機能

- 🔍 キーワード検索
- 📋 ワンクリックでコピー
- 📌 最近使用した顔文字の表示
- 🔥 人気カテゴリー
- 🌐 より多くの顔文字は kaomojiya.org へ

## 📝 カスタマイズ

`data.js` ファイルを編集することで、カテゴリーや顔文字を追加・変更できます。
