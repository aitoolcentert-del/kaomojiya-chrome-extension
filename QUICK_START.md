## すぐにテストする方法（アイコンなし）

アイコンがなくても拡張機能をテストできます！以下の手順に従ってください：

### 1. Chrome で拡張機能を読み込む

1. Chrome ブラウザを開く
2. アドレスバーに `chrome://extensions/` と入力してEnter
3. 右上の **「デベロッパーモード」** をONにする
4. **「パッケージ化されていない拡張機能を読み込む」** をクリック
5. このフォルダ (`/Users/linjiahong/money_making/shipany/chrome-extension/kaomoji`) を選択

⚠️ アイコンがないという警告が出ますが、機能には影響ありません！

### 2. 拡張機能を使う

- Chrome ツールバーの拡張機能エリア（パズルアイコン）をクリック
- 「顔文字屋 - Kaomojiya」を見つけてピン留めすると便利です
- クリックして使ってみてください！

### 3. 後でアイコンを追加する（オプション）

#### 方法A: ブラウザでアイコンを生成（簡単！）

1. `icon-generator.html` をダブルクリックしてブラウザで開く
2. 3つのボタンをクリックして各サイズをダウンロード
3. ダウンロードした画像をこのフォルダに保存
4. Chrome の拡張機能ページで「更新」ボタンをクリック

#### 方法B: Node.js で生成（開発者向け）

```bash
npm install canvas
node generate-icons.js
```

## 🎯 主な機能

- ✅ キーワード検索（例: "笑顔"、"sad"、"bear"）
- ✅ カテゴリーから選択
- ✅ クリックで即コピー
- ✅ 最近使った顔文字を表示
- ✅ 30個以上は「もっと見る」で kaomojiya.org へ

## 💡 使い方のヒント

1. **検索**: "嬉しい"、"love"、"猫" など、日本語でも英語でもOK
2. **最近使用**: 一度使った顔文字は上部に表示されます
3. **カテゴリー**: 12種類のカテゴリーから探せます
4. **もっと見る**: 各カテゴリー30個以上ある場合、ボタンで公式サイトへ

## 🐛 トラブルシューティング

### 拡張機能が表示されない
- デベロッパーモードがONになっているか確認
- フォルダの場所が正しいか確認
- Chrome を再起動してみる

### クリックしてもコピーされない
- ブラウザの設定でクリップボードの権限を確認
- manifest.json に "clipboardWrite" 権限があるか確認

### アイコンの警告が気になる
- icon-generator.html でアイコンを生成してください
- または警告は無視しても機能は正常に動作します

## 📝 次のステップ

- data.js を編集してカテゴリーや顔文字を追加
- popup.css でデザインをカスタマイズ
- MAX_DISPLAY の値を変更して表示数を調整（popup.js:19行目）
